# IPython and Jupyter Notebook

## IPython

IPython is an advanced interactive Python Shell.

### Installation

Installation on Arch Linux:

```
pacman -S ipython
```


### Usage

```bash
ipython -h
```

## Jupyter


### Usage

To start the notebook server run 

```bash
ipython notebook
```

or

```bash
jupyter-notebook 
```

Now open a browser and go to `http://localhost:8888`.

### Shortcuts

Show shortcuts:

```
CTRL + SHIFT + P
```
